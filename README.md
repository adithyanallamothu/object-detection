# Object Detection and Tracking

## Description
Object Detection and Tracking is a Computer Vision project that detects and identifies objects in real-time using a webcam feed. The system uses a pre-trained YOLO model to recognize objects and display bounding boxes with labels around detected items.

## Features
- Real-time object detection
- Live webcam integration
- Automatic object labeling
- Bounding box visualization
- Fast and accurate detection

## Technologies Used
- Python
- OpenCV
- YOLO (You Only Look Once)
- Computer Vision

## How It Works
1. Webcam captures video frames.
2. Each frame is processed by the YOLO model.
3. Objects are detected and classified.
4. Bounding boxes and labels are drawn around detected objects.
5. Results are displayed in real time.
